## 0x00

spotlight 很方便，可以搜索 mac 上绝大部分文件，有时候需要显示文件所在的文件夹，如何操作？

<!--more-->

## 0x01

其实很简单，`command` + `return`(回车) 即可

其他命令：

```sh
command + space # 打开 Spotlight

ctrl + n # 下一个选项

ctrl + p # 上一个选项
```
